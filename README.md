# Youtube Downloader

* Installation

```bash
> apt update && apt upgrade
> apt install git
> apt install nodejs
> git clone https://github.com/RikyXDZ/YoutubeDL.git
```

* Setup

```bash
> cd YoutubeDL
> ./install
```

* Start
```bash
> ./start
```
